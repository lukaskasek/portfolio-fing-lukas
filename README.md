# 💼 Portfolio – Proyectos de Facultad (FING)

Este repositorio reúne una selección de proyectos desarrollados durante mi formación en Ingeniería en Computación en la Facultad de Ingeniería (UdelaR).

El objetivo es mostrar de forma práctica los conceptos trabajados en distintas áreas de la carrera, incluyendo programación, diseño de software, redes y robótica.

---

## 📂 Contenido

El repositorio está organizado por áreas/materias:

* **📡 Redes de Computadoras**
  Implementaciones y trabajos relacionados con comunicación entre sistemas, manejo de protocolos y conceptos de networking.

* **🧮 Programación Funcional**
  Resolución de problemas utilizando paradigmas funcionales, con foco en recursión, manejo de listas y lógica declarativa.

* **💻 Programación Orientada a Objetos**
  Desarrollo de soluciones utilizando principios de diseño OO, encapsulamiento y modelado de sistemas.

* **🧪 Taller de Desarrollo de Software**
  Proyecto más integral enfocado en el desarrollo de una aplicación, incluyendo diseño, implementación y organización del código.

* **🤖 Robótica Autónoma**
  Implementación de comportamientos autónomos, integrando lógica de decisión, sensores y control.

---

## 🛠️ Tecnologías y herramientas

A lo largo de los proyectos se utilizaron distintas tecnologías, entre ellas:

* C / C++ / Java / Python
* Prolog / paradigmas funcionales
* Herramientas de modelado y diseño
* Conceptos de redes y comunicación
* (Según el proyecto) simuladores o frameworks específicos

---

## 🎯 Objetivo del repositorio

Este repositorio no busca ser una colección exhaustiva de entregas, sino una selección curada de trabajos que reflejan:

* Resolución de problemas
* Aplicación de conceptos teóricos en código
* Capacidad de adaptación a distintos paradigmas
* Organización y estructuración de proyectos
